Biotica, a serious game

Biotica is a work in progress project. It's a serious game + an hybrid game using augmented reality, about biodiversity in cities. We wanted to create a game to raise awareness among people about the interaction between the fauna & flora in cities. Players needs to cooperate to instaure a beautiful & useful nature in their cities. The board is modular, and players can choose the tiles they want so each game is different They will learn a lot about the biodiversity while having (i hope) fun during the game.

Currently, we created a POC, with a board game, cards, pieces and an Unity app using Vuforia + a prototype made on adobe XD for the experience.
We meet a lot of people, just to gather the right informations about the flora. We made some users play test, during a board game festival. The game will go through many changes and maybe, it won't be the same in the future.
