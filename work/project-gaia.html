Project Gaia

This project is still WIP, it needs a full redesign. At first, it was a UI exercise.
Brief : Design a full-colour mock-up of a loadout menu for an online first-person shooter.
Must have :
- Categories of equipable items - inventory grid of one of the item categories above
- Show states for buttons: idle, hover, down, disabled
- Show stats and details about each item.
- Indicate which item is currently equipped.
- Space for a 3D character to be displayed
- Navigation to other parts of the UI (Profile, Social, Settings, etc.)
- A layout that can accommodate both PC and Console control schemes.

I will remake pretty much everything, layout, typography, style to make it unique.
Not another uninspired Scifi First Person Shooter with no uniqueness what so ever.
A complete brand new art direction, based on SolarPunk & bio-enhancements with some low-tech elements.
Maybe it will be a low-tech interface with dither and minimalist elements. I will try to make something different & unique.
