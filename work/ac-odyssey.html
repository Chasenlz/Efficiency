AC Odyssey UI Redesign

For this personal project, I imagined a redesign of the inventory menu. It was only a exercise, not saying the actual UI has issues, quite the opposite, the UI is clean and the art direction is really beautiful. I only wanted to tweak the UI. My main inspiration for this redesign is the inventory of The Witcher 3, which allow the player to see every item he owns. The player won't have to enter inside a item selection to see if he has a better item to equip.

Also, I added some indicators to the shortcut menu, to reduce the player's cognitive load. The actual shortcut menu is great, especially on console, when you press the RT button, the other menu shortcut minimize.
